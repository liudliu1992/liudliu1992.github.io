deepin 屏幕闪烁解决方案

    这个系统在滑动的时候, 屏幕一直闪烁, 我也百度到了一篇帖子, 也是同样的问题, 然后一路找到了 github 上, 并根据相关回复, 顺利解决了问题.
    我的系统: deepin 15.11 桌面版, 安装KDE系统设置工具 sudo apt install systemsettings, 然后打开这个软件, 翻到最下面的 显示和配置, 点进去将Redering backend 改为 XRender, 确认应用即可.
